## 容器化的底层原理

* 依赖 Linux Container 技术 ，核心两个组件 Namespace、Cgroups

### Namespace

* 提供资源隔离，定义全新运行环境的能力。linux kernel clone 方法，容器本质还是一个进程

### Cgroups

* 提供限制资源使用上限的能力。例如：CPU、内存、磁盘等
