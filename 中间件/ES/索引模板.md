## 索引模板

通过表达式匹配索引，在创建索引时定义字段影射规则。

```
"template_name" : {
    "order" : 0,
    "index_patterns" : [
      "report_consumer_*",
      "report_synchronizer_*"
    ],
    "settings" : {
      "index" : {
        "lifecycle" : {
          "name" : "ndhc-report-history"
        }
      }
    },
    "mappings" : {
      "properties" : {
        "reportTimestamp" : {
          "type" : "date"
        }
      }
    },
    "aliases" : { }
  }
```
